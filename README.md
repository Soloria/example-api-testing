# Example for testing Google Translate API on Typescript

### Dependencies

- Yarn 1.6 or above   
- NodeJS 9.0 or above    
- TypeScript 2.6 or above (in global)    
- Mocha 5.1 or above (in global)     
  
### Install&Run

- Create config.ts  
- Copy class for config.example.ts to config.ts   
- Set API-key  
- Run `Run.ps1` or `yarn test`
